## COMANDOS UTILIZADOS 
**1**: PWD se utiliza al comienzo para saber donde estamos parados, en que directorio.
**2**: CD comando para ingresar a una carpeta determinada, ej:ingrese al directorio donde quiero crear archivos en mi caso DESKTOP.
**3** MKDIR sirve para crear carpetas, cree mi primera carpeta tp1-mi-sitio
**4**: LS genera un listado en base a los archivos que hayan en una carpeta
**5**: TOUCH genera archivos, con este comando cree mi primer index y style.css
**6**: code. sirve para abrir el VSC
**7**: cat lo utilize para ver el contenido de mi archivo index
**8**:cd .. lo utilize para volver a la carpeta tp1-mi-sitio para crear otro archivo
**9**: CLEAR con este comando limpie la terminal y comence a practicar nuevamente la creacion de carpetas y archivo
**10**: which para corroborar donde se instalo git
